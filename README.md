# Quick-Calc

Quick-Calc is a simple calculator application built with JavaScript.
It supports addition, subtraction, multiplication, and division.
The project demonstrates software testing and version control using GitHub.

## Setup

Install dependencies:

npm install

## Run Tests

npm test

## Testing Framework Research

This project uses **Jest** as the testing framework.

Jest is a popular JavaScript testing framework that provides built-in assertions,
mocking, and test runners. It is easy to configure and widely used in Node.js
projects.

Another popular testing framework is **Mocha**. Mocha is flexible and powerful
but often requires additional libraries like Chai for assertions.

Jest was chosen because it has a simple setup and provides all testing features
in a single package.
